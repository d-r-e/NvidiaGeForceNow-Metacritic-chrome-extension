# Privacy Policy

GeForce NOW Metacritic Score does not collect, sell, or transmit personal data.

## What the extension accesses

- `https://play.geforcenow.com/*` to read visible game titles and place score badges in the page UI.
- `https://www.metacritic.com/*` to request Metacritic search results for those game titles.
- `chrome.storage.local` to cache resolved matches and scores on the user's device.

## What the extension stores

The extension stores a local cache of matched titles and scores in the browser so it does not need to repeat the same lookups on every page visit.

## What the extension does not do

- collect account credentials
- collect payment information
- track browsing outside the declared host permissions
- send analytics or telemetry
- create remote user profiles
